# ECMAScript6

# let 和 const

  • ES6 新增了 let 和 const 来声明变量，主要是解决 var 声明变量所造成的困扰和问题

    • var 不能用于定义常量

    • var 存在变量提升

  • let

    • 不能重复声明变量

    • 支持块级作用域

  • 注意

    • let 和 const 声明的变量不会放在 window 上

# Map

  • Map 结构提供了 “值 - 值” 的对应，是一种完善的 hash 结构实现，如果你需要键值对的数据结构，Map 比 Object 更合适，他类似与对象，也是键值对的集合，但是 “键” 的范围不止于字符串，各种类型的值（包括对象）都可以当做键
